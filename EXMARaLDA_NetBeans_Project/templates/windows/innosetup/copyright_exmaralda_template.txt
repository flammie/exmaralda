EXMARaLDA tools are free to use. EXMARaLDA's source code is licensed under an open source license and available from GitHub (https://github.com/EXMARaLDA/exmaralda/).

The tools are developed by Thomas Schmidt (IDS Mannheim) and Kai Woerner (University of Hamburg) with support from Hanna Hedeland, Daniel Jettka (HZSK Hamburg) and other contributors.

Work on EXMARaLDA was and is funded by the German Research Association (DFG), the Bundesministerium fuer Bildung und Forschung (BMBF), the University of Hamburg, and the Institut fuer Deutsche Sprache (IDS).

Localisations were done with support from Annette Schnieders (German), Tulay Selcuk (Turkish), Hanna Hedeland (Swedish), Petr Porizka (Czech), Silke Merkel (Spanish) and Fan Junjun (Chinese).

EXMARaLDA uses the following libraries:
* parts of the ELAN code (http://www.lat-mpi.eu/tools/elan/)
* parts of the BAS code (http://www.phonetik.uni-muenchen.de/forschung/bay_arch_sprsig/index.html)
* TreeTagger for Java (https://www.openhub.net/p/tt4j)
* JDOM (http://www.jdom.org/)
* FOBS for JMF (http://fobs.sourceforge.net/)
* Quaqua (http://www.randelshofer.ch/quaqua/)
* Saxon, Xalan, Bounce and Cobra.
